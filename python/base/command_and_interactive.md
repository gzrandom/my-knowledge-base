
# Python命令行模式&交互模式

### 命令行模式

在系统命令行执行python文件

`c:\python hello_world.py`

不会自动返回值，需要使用print()。


### 交互模式

命令行使用python进入交互模式
```
   c:\python
   >>>
```
每行输入的命令都会立即执行并返回结果

### 运行python文件
1 win
   
   `python hello.py`
   
2 linux
   * $python hello.py
   * 直接运行
      在.py文件第一行添加
      
      ` #!/usr/bin/env python3`
      
      添加执行权限
      
      `chmod a+x hello.py`
